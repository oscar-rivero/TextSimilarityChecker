About Plagiarism Detector

This application helps detect potential plagiarism by comparing submitted text with online sources.

Features include:

  • Text similarity analysis
  • Matching phrase detection
  • Visual representation of results
  • Report generation

Built with Django, NLTK, and modern web technologies.
